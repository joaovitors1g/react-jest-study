## Executando

Primeiramente faça o seguinte

```bash
$ git clone https://github.com/joaovitors1g/react-jest-study.git
$ cd react-jest-study
$ npm i
```

### Testes

```bash
$ npm test
```

### Code Coverage

```bash
$ npm run coverage
```

Os arquivos aparecerão na pasta `src/__tests__/coverage`
